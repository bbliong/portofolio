DKDEPOKDocs

Quick Start

Last updated: July 19th, 2018

Permulaan

Ini merupakan dokumentasi dari API Diskominfo informasi Infrastruktur Sarana dan Prasarana Sekolah Se-Kota Depok. teknologi yang digunakan adalah Laravel Lumen dengan tingkat authentikasi JWT Auth.

Status kode yang akan dikeluarkan
# Status Kode Status Deskripsi Solusi
1 200 Success Tidak terdapat masalah -
2 200 error Terjadi kesalahan dengan data Lihat pesan error
4 401 error Tidak terautorisasi Login kembali
5 500 error Kesalahan server Laporkan ke admin

Autentikasi

Sign Up

Digunakan untuk mendaftarkan User baru pada database DKDEpok.

POST http://dkdepok.xyz/api/signup

Payload :
{
     "username": "frederikStef",
     "name": "frederik",
     "email": "frederik@gmail.com",
     "password" : "qwerty123",
     "password_confirmation" : "qwerty123",
     "school_id"   : 24
}
Output :
 {
    "username": "frederikStef",
    "name": "frederik",
    "email": "frederik@gmail.com",
    "school_id": 24,
    "updated_at": "2019-06-19 09:06:56",
    "created_at": "2019-06-19 09:06:56",
    "id": 12
}

Perlu diperhatikan, sebelum mendaftar harus terlebih dahulu mengetahui id dari sekolah yang tersedia pada database, jika tidak maka tidak akan bisa mendaftar

Sign In

DKDepok API menggunakan JWT Token untuk melakukan autentikasi. Hampir seluruh endpoint yang ada pada API ini harus melalui autentikasi terlebih dahulu.

Ketika melakukan autentikasi maka akan diberikan suatu token yang terdiri dari alphanumerik lebih dari 100 karakter, simpan token ini dan ketika ingin mengakses end point lainnya perlu ditambahkan header Authorization ketika melakukan request

position http://dkdepok.xyz/api/signin

Payload :
 {
    "email" : "frederik@gmail.com",
    "password" : "qwerty123"
}
Output :
